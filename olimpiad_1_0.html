/contest/{constestID}/tasks/{id} get

get

Возвращает текст задачи id

get /contest/{constestID}/tasks/{id}

Возвращает текст задачи id

  • Request
  • Response

URI Parameters

  • constestID: required (string)
  • id: required (string)

HTTP status code 200

Задание доступно

Body

Media type: application/json

Type: object

Properties
  • header: required (string)

    Заголовок задачи

  • body: required (string)

    Условие задачи

  • progress: required (object)

    описывает прогресс пользователя в решении

    • code: required (integer)

      состояние (см в трелло в карточке Коды состояния решения)

    • tests_passed: required (integer)

      количество пройденных тестов

    • tests_count: required (integer)

      общее число тестов

  • pictures: (array of string)

    Иллюстарции к задаче (если имеются)

  • examples: required (array of object)

    Items: items

    • input: required (string)
    • output: required (string)

Media type: example

Type:

{
  "header": "Задача",
  "body": "Многобуквенный текст задачи..............",
  "progress": {
      "code": 1,
      "tests_passed": 4,
      "tests_count": 4
  },
  "pictures": [
    "https://mondrian.mashable.com/2016%252F09%252F22%252Fb5%252F592857175_640.73223.jpg%252F1200x630.jpg?signature=R9W_YTladVpWShfFUQ4Ja6vLZI4="
  ],
  "examples": [
    {
      "input": "1 2 3 4 5 6",
      "output": "YES"
    }
  ]
}
